# My AstroNvim config

### How to:
#### Clone the repository to nvim folder

```shell
git clone https://github.com/Georgefwm/astro-config $env:LOCALAPPDATA\nvim
```

#### Remove git directory

```shell
Remove-Item $env:LOCALAPPDATA\nvim\.git -Recurse -Force
```

#### Start Neovim

```shell
nvim
```
